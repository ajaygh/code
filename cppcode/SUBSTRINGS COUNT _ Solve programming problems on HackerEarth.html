SUBSTRINGS COUNT

Jack is the most intelligent student in the class.To boost his intelligence,his class teacher gave him a problem named "Substring Count".

Problem :
His Class teacher gave him n strings numbered from 1 to n which consists of only lowercase letters (each having length not more than 10) and then ask Q questions related to the given strings.

Each question is described by the 2 integers L,R and a string str,Now his teacher wants to know how many strings numbered from L to R contains str as a substrings.

As expected, Jack solved this problem with in a minute but he failed to solve it efficiently.Now ,its your turn to teach him.How to do it efficiently?? and save him from punishment.

INPUT
First line of input contains a single integer n denoting the number of strings given by class teacher to jack.Next n lines of input contains n strings (one per line).Next line fo input contains a single integer Q denoting the number of questions asked by his teacher.next Q lines of input contains Q question (one per line) as explained above.

OUTPUT
print the correct answer for each of the question asked by his teacher.

CONSTRAINTS
1<=n<=10000
1<=strlen(str)<=10
1<=Q<=5*10^5
1<=L,R<=n

NOTE: strings consist of only lower case characters

SAMPLE INPUT
3
code
coder
coding
2
1 3 code
1 3 co
SAMPLE OUTPUT
2 
3
Explanation

Q1:: code coder coding only two out of 3 strings contain cod as substring Q2:: code coder coding all the 3 strings contain co as substring

Time Limit: 5 sec(s) for each input file.
Memory Limit: 256 MB
Source Limit: 1024 KB
Marking Scheme: Marks are awarded if any testcase passes.
Allowed Languages: C, CPP, CLOJURE, CSHARP, GO, HASKELL, JAVA, JAVASCRIPT, JAVASCRIPT_NODE, LISP, OBJECTIVEC, PASCAL, PERL, PHP, PYTHON, RUBY, R, RUST, SCALA

Problem Editorialist: Ganesh Sawhney

Prerequisites: STL

Problem:

We are given a set of N strings numbered (1 to N).Let us signify this set by G(1,N).

Then we are given Q queries where each query has L, R and a String S.

Now for each query we need to find - How many strings in the set G(L,R) have S as a substring?

Solution:

The Obvious brute solution: The most obvious way to solve the problem would be to store G(1,N) in a string array.

And Then for each query, we search S as substring in the set G(L,R).

But this solution will give TLE in TESTCASES – 3, 4, 5.

The Faster solution using STL::Map:

A Faster approach would be to use Map and Vectors. We divide the solution into 3 Operations.

Operation – 1:

Initially we create a map M with string as the Key and vector as the value. { map > m; }

For each string(i) in G(1,N), we store all the possible substrings in the map (with the key being the substring) and push the

index(i) in the value vector.

Example let the string(i) be “code” with its index i=1. The corresponding map entries will be:

c => 1

co => 1

cod => 1

code => 1

o => 1

od => 1

ode =>1

d => 1

de => 1

e => 1

Operation – 2:

Now a small issue arises. Suppose a string is “codecode”, here many map entry will get two times ‘1’ in the value vector.

For eliminating these duplicate entries, we iterate for every map key and apply “unique”, “resize” operations to the value

vector.

Operation – 3:

Now we tackle each query Q. Let for ith query the string given be S. Thus we count the total number of entries between L

and R in the vector “m[S]” using upper_bound and lower_bound operations.

Note: We use scanf and printf rather than using cin, cout for faster input and output.

#include <bits/stdc++.h>
using namespace std;
string getstr() //To Input String
{ 
    char a[11];
    scanf("%s",a);
    string s=a;
    return s;
}

int main()
{
    int n;
    scanf("%d",&n);
    //Operation - 1
    map <string, vector<int> > m;
    string s;
    for ( int i = 0; i < n; i++ ) 
    {
        s=getstr();
        for ( int j = 0; j < (int)s.size(); j++ )
        {
            string s1("");
            for ( int k = j; k < (int)s.size(); k++ ) 
            {
                s1.push_back(s[k]);
                m[s1].push_back(i);
            }
        }
    }

    //Operation - 2
    for(map <string, vector<int> >::iterator it = m.begin(); it != m.end(); ++it)
    {
        vector<int>::iterator sz = unique((it->second).begin(),(it->second).end());
        (it->second).resize(distance((it->second).begin(),sz));
    }

    int q;
    scanf("%d",&q);

    //Operation - 3
    while ( q-- )
    {
        int l,r;
        scanf("%d %d",&l,&r);
        s=getstr();
        l--; r--;
        int ans=0;
        if ( s.empty() )
            ans = r-l+1;
        else
        {
            int idx2 = upper_bound(m[s].begin(),m[s].end(),r) - m[s].begin();
            int idx1 = lower_bound(m[s].begin(),m[s].end(),l) - m[s].begin();
            ans = idx2-idx1;
        }
        printf("%d\n",ans);
    }
    return 0;
}
